# 🛡️ WhiteVPN Suite

**WhiteVPN Suite** — современный высокопроизводительный VPN-клиент и веб-панель управления на Node.js, созданный для обхода жестких блокировок ТСПУ, интеграции протоколов VLESS / VLESS Reality / XHTTP, поддержки Wintun системного туннелирования и Двойного VPN (Double VPN Chaining).

---

## ✨ Основные Возможности

- 🚀 **Поддержка Протоколов нового поколения**:
  - **VLESS / VLESS Reality** (XTLS Vision)
  - **VLESS XHTTP**
  - **Shadowsocks / Trojan**
- 🛡️ **Двойной VPN (Double VPN Chaining)**:
  - Автоматическое объединение транзитных реле-узлов РФ с зарубежными нодами выхода.
- ⚡ **Два Режима Туннелирования**:
  - **Wintun (Sing-Box)**: Полноценная виртуальная сетевая карта на уровне ядра Windows для 100% защиты всего ПК/приложений.
  - **System Proxy / WinINet (Xray)**: Высокоскоростной прокси-режим без установки сетевых драйверов.
- 📊 **Автоматический Пинг и Сортировка Узлов**:
  - Автоматическая сортировка серверов в списке по минимальному URL-пингу и задержке.
- 🎨 **Современный Glassmorphism UI**:
  - Красивый адаптивный веб-интерфейс с темной темой, динамическими карточками статуса и онлайн-графиками трафика.

## 🛠️ Быстрый Запуск

### 1. Требования
- **Windows 10 / 11** (64-bit)
- **Node.js** v18+ 
- Права Администратора (для запуска Wintun адаптера)

### 2. Установка зависимостей
```bash
npm install
```

### 3. Запуск
Для полноценной работы Wintun туннеля запустите проект от имени Администратора через `start-as-admin.bat` или команду:
```bash
npm start
```
После запуска откройте браузер по адресу: `http://localhost:3000`.

В данном репозитории **отсутствуют какие-либо приватные подписки, серверные UUID или секретные ключи**. Пользователи вносят свои подписки V2Ray/VLESS непосредственно через веб-интерфейс приложения.
